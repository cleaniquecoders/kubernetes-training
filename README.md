- [Hello World](https://github.com/cleaniquecoders/docker-php-hello-world.git)

## Usage

Make sure to install [minikube](https://minikube.sigs.k8s.io/docs/start/).

Start minikube:

```bash
minikube start
```

Run the dashboard:

```bash
minikube dashboard
```
